# 基础数学

## 辗转相除法证明

<!-- notecardId: 1736566474068 -->

**辗转相除法**（欧几里得算法）的正确性可以通过数学归纳法和数论中的基本性质来证明。以下是详细的证明过程：

---

### 1. **问题描述**

给定两个整数 $$a$$ 和 $$b$$（假设 $$a > b$$），辗转相除法的目标是找到它们的最大公因数 $$\gcd(a, b)$$。

---

### 2. **辗转相除法的核心思想**

辗转相除法的核心思想是基于以下性质：

$$
\gcd(a, b) = \gcd(b, a \mod b)
$$

其中，$$a \mod b$$ 表示 $$a$$ 除以 $$b$$ 的余数。

---

### 3. **证明 $$\gcd(a, b) = \gcd(b, a \mod b)$$**

我们需要证明：对于任意两个整数 $$a$$ 和 $$b$$（$$a > b$$），$$\gcd(a, b)$$ 等于 $$\gcd(b, a \mod b)$$。

#### 证明步骤：

1. **设 $$d = \gcd(a, b)$$**：

   - 根据最大公因数的定义，$$d$$ 是 $$a$$ 和 $$b$$ 的最大公因数，即 $$d$$ 能整除 $$a$$ 和 $$b$$。

2. **设 $$r = a \mod b$$**：

   - 根据除法的定义，$$a = b \cdot q + r$$，其中 $$q$$ 是商，$$r$$ 是余数，且 $$0 \leq r < b$$。

3. **证明 $$d$$ 是 $$b$$ 和 $$r$$ 的公因数**：

   - 因为 $$d$$ 能整除 $$a$$ 和 $$b$$，所以 $$d$$ 也能整除 $$a - b \cdot q$$，即 $$d$$ 能整除 $$r$$。
   - 因此，$$d$$ 是 $$b$$ 和 $$r$$ 的公因数。

4. **证明 $$d$$ 是 $$b$$ 和 $$r$$ 的最大公因数**：

   - 假设存在另一个整数 $$d'$$，使得 $$d'$$ 是 $$b$$ 和 $$r$$ 的公因数，且 $$d' > d$$。
   - 因为 $$d'$$ 能整除 $$b$$ 和 $$r$$，所以 $$d'$$ 也能整除 $$a = b \cdot q + r$$。
   - 这意味着 $$d'$$ 是 $$a$$ 和 $$b$$ 的公因数，且 $$d' > d$$，这与 $$d$$ 是最大公因数的定义矛盾。
   - 因此，$$d$$ 是 $$b$$ 和 $$r$$ 的最大公因数。

5. **结论**：
   - $$\gcd(a, b) = \gcd(b, r)$$，即 $$\gcd(a, b) = \gcd(b, a \mod b)$$。

---

### 4. **终止条件的证明**

辗转相除法的终止条件是 $$b = 0$$。此时，$$\gcd(a, 0) = a$$，因为任何数和 0 的最大公因数都是它本身。

---

### 5. **算法正确性的总结**

通过上述证明，我们可以得出：

- 辗转相除法通过反复应用 $$\gcd(a, b) = \gcd(b, a \mod b)$$，将问题规模不断缩小。
- 当 $$b = 0$$ 时，算法终止，此时的 $$a$$ 就是最大公因数。
- 由于每次迭代至少将问题规模减半，算法的时间复杂度为 $$O(\log(\min(a, b)))$$。

---

### 6. **示例验证**

以 $$a = 56$$ 和 $$b = 98$$ 为例，验证辗转相除法的正确性：

1. 第一次迭代：

   - $$a = 98$$, $$b = 56$$。
   - $$98 \mod 56 = 42$$。
   - $$\gcd(98, 56) = \gcd(56, 42)$$。

2. 第二次迭代：

   - $$a = 56$$, $$b = 42$$。
   - $$56 \mod 42 = 14$$。
   - $$\gcd(56, 42) = \gcd(42, 14)$$。

3. 第三次迭代：

   - $$a = 42$$, $$b = 14$$。
   - $$42 \mod 14 = 0$$。
   - $$\gcd(42, 14) = \gcd(14, 0)$$。

4. 终止条件：
   - $$b = 0$$，返回 $$a = 14$$。
   - 因此，$$\gcd(56, 98) = 14$$。

---

### 7. **数学归纳法证明**

为了更严谨地证明辗转相除法的正确性，可以使用数学归纳法。

#### 归纳假设：

假设对于所有 $$k < n$$，辗转相除法能够正确计算 $$\gcd(a, b)$$。

#### 归纳步骤：

1. 对于 $$k = n$$，设 $$a = b \cdot q + r$$，其中 $$0 \leq r < b$$。
2. 根据归纳假设，$$\gcd(b, r)$$ 能够正确计算。
3. 根据 $$\gcd(a, b) = \gcd(b, r)$$，辗转相除法能够正确计算 $$\gcd(a, b)$$。

#### 终止条件：

当 $$b = 0$$ 时，$$\gcd(a, 0) = a$$，算法终止。

---

### 8. **总结**

- 辗转相除法的正确性基于 $$\gcd(a, b) = \gcd(b, a \mod b)$$ 的性质。
- 通过数学归纳法和数论中的基本性质，可以严格证明算法的正确性。
- 该算法高效、简单，适用于任意大小的整数。
